_A Nim wrapper to reduce your link._

### How to use
```bash
nimble build -d:ssl
shortnim --provider <service> <url>
```
Example:
```bash
shortnim -p relink https://github.com/Owanesh
shortnim -p cleanuri https://duckduckgo.com/
```

### Works with
* [Rel.ink](https://rel.ink/)
* [CleanURI](https://cleanuri.com/)
